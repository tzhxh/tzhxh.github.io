关于post第一次失败，后面可以成功的问题记录

发表于 2019-11-03

在使用封装的ajax进行与后台数据传输时发现的一个问题

第一次post总是会被挂起，status状态为cancel。只有再发送才能成功。
经过在网上查找资料并通过实验得出结论

  1. 方法在a标签内触发会导致该问题发生
  2. 方法在from表单内使用button标签触发 不写明type属性，会导致失败，猜测可能是不写明type属性就默认问submit。
  3. 方法在form内使用button标签触发 type属性为button 成功
  4. 方法在form外使用button标签触发 不写明type属性 成功
  5. 方法在form外使用button标签触发 type属性为button 成功

因此猜测可能是在form表单内使用submit，会导致表单提交。 因此和ajax产生冲突，导致ajax被中断。
