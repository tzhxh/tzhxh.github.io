jsp+servlet实现简单的登录跳转功能

发表于 2019-04-20
用jsp和servlet实现一个简单的登录跳转功能的页面

主要流程

1.首先在Tomcat的webapps文件夹下创建一个新的文件夹，并将jsp文件存放在该目录下 这里我将文件夹命名为login 2.在login文件夹下再新建一个名为WEB-INF的文件夹，用来存放命名为web.xml 的配置文件， 3.在WEN-INF文件下再新建一个名为classes的文件夹，用来存放servlet文件 (若servlet文件有包，则文件结构为classes - 包名(文件夹) - servlet文件名.class) 4.将编写好的 .java编译为.class 文件后放入指定目录，并修改配置文件

简单示例

实现简单的登录跳转功能，首先进入到登录页面，在登录页面输入用户名，密码。点击提交。 跳转到loginServlet中判断用户名和密码是否正确，若正确则跳转到工作页面，若不正确跳转回登录页面 共有4个文件组成 分别为 登录页面(login.jsp) 工作页面(page.jsp) servlet文件(LogonServlet.java) 和配置文件(web.xml) 代码如下：

LoginServlet.jsp

一个简单的判断跳转的servlet，逻辑为只要用户名或者密码为admin的都可以登录

点击显示/隐藏代码

package loginServlet;

import java.io.*;
import javax.servlet.ServletException;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
//import javax.servlet.http.HttpSession;

public class LoginServlet extends HttpServlet{

private static final long serialVersionUID = 1L;    
    
protected void doPost(HttpServletRequest request,HttpServletResponse response) throws ServletException,IOException{
    String name = request.getParameter("username");
    String word = request.getParameter("password");
    if("admin".equals(name)||"admin".equals(word))
    {
        response.sendRedirect("page.jsp?name="+name);
    }
    else
    {
        request.getRequestDispatcher("login.jsp").forward(request, response);
    }
}
}

web.xml

配置文件 头部可以在webapps文件夹下的manager中找到它的web.xml进行复制 在 servlet 标签中的是配置文件的主要部分 两个servlet-name可以任意取名 但必须要一致 servlet-class中的内容为存放的路径 这里表示loginServlet包 下的LoginServlet.class文件 .class后缀要省去 url-pattern 是自己填写的路径名 可以随意填写 但注意最左侧的"/"不能缺省 在浏览器中打开时输入 localhost:8080/工程文件夹/自定义的url-pattern

点击显示/隐藏代码

<?xml version="1.0" encoding="UTF-8"?>

<web-app xmlns="http://xmlns.jcp.org/xml/ns/javaee"
  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://xmlns.jcp.org/xml/ns/javaee
                      http://xmlns.jcp.org/xml/ns/javaee/web-app_4_0.xsd"
  version="4.0"
  metadata-complete="true">

<servlet>

      <servlet-name>loginServlet</servlet-name>

      <servlet-class>loginServlet.LoginServlet</servlet-class>
  
</servlet>

<servlet-mapping>

     <servlet-name>loginServlet</servlet-name>

     <url-pattern>/loginServlet</url-pattern>

</servlet-mapping>
</web-app>

展示

在Tomcat的bin文件夹下找到startup.bat文件双击即可开启服务 开启服务后在浏览器中输入路径http://localhost:8080/login/login.jsp 其中localhost是本地的ip，8080为Tomcat的默认端口，webapps下的工程文件夹的名字(根据自己建立的文件夹来修改),login.jsp是要打开的页面(根据自己的名称修改) 结束时双击bin文件夹下的shutdown.bat来关闭服务
